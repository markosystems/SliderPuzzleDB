# SliderPuzzleDB
Every Possible Position:
For some reason there isn't public directory of every solvealbe slider puzzle starting position, I created one.

I'll only be doing perfect squares, because they're easier for scripting.

The script for finding positions, starts at the solved position and makes random moves until it finds the all of the possible positions. It takes a while, and could probably use some optimization.

Feel free to use whatever you want.
